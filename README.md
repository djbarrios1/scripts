# 🛠️ Windows Scripting Toolkit

This repository serves as a central hub for my collection of PowerShell (`.ps1`) and Batch (`.bat`) scripts, created to support automation, administration, and troubleshooting tasks across Windows environments.

These scripts are designed to streamline common IT workflows, enhance system management, and support secure, scalable operations in enterprise and hybrid infrastructure.

## 🔧 What's Inside

- **PowerShell Scripts** – For tasks like software updates, Office 365 management, user provisioning, scheduled jobs, and log analysis  
- **Batch Files** – For legacy automation, simple utilities, and system startup tasks  
- **Environment Focus** – Primarily tailored for Windows Server, Microsoft 365, and domain-joined environments, with emphasis on GPO, Active Directory, and automation  
- **Coming Soon: Bash Scripts** – Cross-platform tools for Linux systems and WSL workflows will be added as the project grows

## ✅ Use Cases

- Automating system updates and software deployment  
- Managing users, permissions, and groups  
- Improving login scripts and startup processes  
- Running diagnostics and gathering system info  

## 📁 Folder Structure

/PowerShell - All .ps1 scripts organized by function /Batch - All .bat files for automation or admin tools /Utils - Shared tools or supporting assets /Docs - Optional documentation and usage notes


## 📌 Notes

- Scripts are regularly updated as environments evolve  
- Where applicable, comments within the scripts explain usage and any required parameters  
- Bash shell scripts for Linux/WSL environments will be added over time  
- Feedback, improvements, or contributions are always welcome
